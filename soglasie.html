Согласие на обработку персональных данных

1. Общие положения

1.1. Настоящим даю согласие на обработку моих персональных данных.

1.2. Обработка данных осуществляется в соответствии с ФЗ-152.

2. Цель обработки

2.1. Оказание медицинских услуг.

2.2. Информирование об услугах Сервиса.

3. Перечень данных

3.1. ФИО, контактный телефон, адрес.

3.2. Медицинская информация (только для оказания услуг).

4. Гарантии конфиденциальности

4.1. Данные не передаются третьим лицам.

4.2. Медицинская информация конфиденциальна.

5. Отзыв согласия

5.1. Согласие может быть отозвано в любой момент.
